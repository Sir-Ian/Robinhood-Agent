# Thesis Files

Each promoted candidate must have a thesis file named `SYMBOL.md` in this directory before it can enter `promoted_universe.jsonl`.

Use this exact structure:

```markdown
# SYMBOL Thesis

## Symbol

## Asset Type

## Date Opened

## Status

## Thesis Summary

## Evidence

## Pattern/Signal Observed

## Catalyst

## Why This Is Better Than The Current Funding Source

## Position Sizing Rationale

## Entry Logic

## Exit Logic

## Invalidation Logic

## Max Loss Assumption

## Expected Reward/Risk

## What Would Prove The Thesis Wrong

## Bear Case Review

### Main Bear Case

### What Would Disprove The Thesis

### What Could Go Wrong Operationally

### Liquidity/Spread Concern

### Event/Earnings Concern

### Why The Trade Is Still Worth Considering

### Exit Trigger

## Monitoring Checklist

## Sources Checked

## Last Updated
```

The opportunity researcher owns thesis creation and updates. The capital allocator may reference theses but should not weaken invalidation, exit, or risk language to force a trade.
